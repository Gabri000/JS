Assignement 8 : Using Arrays

Description:
  1. Create a new HMTL page for the exercise as normally.
  2. Declare an array that contains 10 friends of yours. Example: var myArray = ["blue", "pink", "white"];
  3. Use the command document.write(xxx) to print the 3rd name in the array.
  4. Use the syntax myArray[x] = "newName" to change the 3rd name in the array.
  5. Use the command document.write(xxx) to print the changed name.
  6. Remember to add a link to the index.html that teacher can see it.
